WorkGains Website Overview
Purpose: Designed for individuals interested in daily food intake and activity logs.
Pages: Includes Home, About Us, Store, Quiz, Newsletter, Gallery, My List, and Favorite Page.
Features:
The Quiz page is implemented with JavaScript, with a timer and a scoring system where correct answers earn 2 marks and wrong answers deduct 1 mark. Badges (Gold, Silver, Bronze, or Fail) are awarded based on the final score.
The Products Page uses JavaScript and CSS for an 'add to cart' process, allowing up to 20 items in the cart.
The Favorite Page allows users to change the background and text colors using JavaScript functions.
The My List Page includes two buttons to show a Leaderboard of top points holders by loading XML data using XMLHttpRequest() and DOMParser() objects.

UX/UI and Accessibility
Design: The primary colors are Black, Yellow, and White. Color contrast tests show compliance with WCAG 2 AA and AAA standards for several color combinations.
Accessibility: Techniques include setting the HTML title and language, using tags for screen readers, using alternate text for images, and using <label> for form elements. The WAVE accessibility evaluation tool reported 1 error and 2 alerts across several pages.
